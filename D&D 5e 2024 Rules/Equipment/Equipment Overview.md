## Carrying Capacity

### Base Carrying
**Carrying Capacity** = Strength score × 15 pounds

### Push, Drag, Lift
**Maximum** = Strength score × 30 pounds

### Size Modifiers
| Size | Multiplier |
|------|-----------|
| Tiny | ×0.5 |
| Small | ×1 |
| Medium | ×1 |
| Large | ×2 |
| Huge | ×4 |
| Gargantuan | ×8 |

### Encumbrance (Optional)
| Weight | Effect |
|--------|--------|
| > STR × 5 | Speed -10 ft. |
| > STR × 10 | Speed -20 ft., disadvantage on physical checks/saves |

## New in 2024: Weapon Mastery

Certain classes (Barbarian, Fighter, Paladin, Ranger, Rogue) unlock special properties on weapons they're proficient with.

| Mastery | Effect |
|---------|--------|
| Cleave | Extra attack vs. adjacent creature |
| Graze | Deal STR/DEX mod damage on miss |
| Nick | Extra attack without Bonus Action |
| Push | Push target 10 ft. |
| Sap | Target has disadvantage on next attack |
| Slow | Reduce target's speed by 10 ft. |
| Topple | Target must save or fall Prone |
| Vex | Advantage on next attack against target |

See [[Weapons]] for which weapons have which Mastery.
