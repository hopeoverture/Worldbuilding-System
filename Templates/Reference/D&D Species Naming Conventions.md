# D&D Species Naming Conventions

> [!info] Reference Guide
> Naming patterns for standard D&D species to maintain cultural authenticity and linguistic consistency. Use these patterns when generating characters, NPCs, and settlements associated with specific species.

## Human Names

Humans have the most diverse naming conventions, typically reflecting real-world cultural inspirations.

### Common Cultural Patterns
| Culture Style | Male Examples | Female Examples | Surname Style |
|---------------|---------------|-----------------|---------------|
| Germanic | Wilhelm, Aldric, Konrad | Greta, Brunhilde, Ingrid | Occupational (Miller, Smith) |
| Latin/Romance | Marcus, Lucius, Anton | Julia, Livia, Celeste | Family names (-ius, -us) |
| Celtic | Brennan, Cormac, Declan | Aisling, Brigid, Niamh | Patronymic (Mac-, O'-) |
| Slavic | Dmitri, Vasili, Nikolai | Anastasia, Svetlana, Yelena | -ov, -ev, -sky endings |
| Norse | Erik, Bjorn, Ragnar | Freya, Astrid, Sigrid | -son, -dottir patronymics |
| Arabic | Khalid, Omar, Rashid | Fatima, Layla, Zara | Family/tribal names |
| East Asian | Kenji, Wei, Hiroshi | Mei, Yuki, Sakura | Family name first |

### Naming Tips
- Match naming culture to the region's inspiration
- Surnames often reflect occupation, location, or ancestry
- Nobility may have compound surnames or titles

## Dwarven Names

Dwarven names feature **harsh consonants**, **guttural sounds**, and strong syllables that evoke stone and metal.

### Phonetic Elements
- **Common consonants:** K, G, D, T, R, N, TH, B
- **Common vowels:** Short A, I, O, U (rarely long vowels)
- **Endings:** -in, -im, -ur, -ak, -or, -ek, -ir

### Male Names
| Name | Meaning/Notes |
|------|---------------|
| Thorin | "Bold one" |
| Balin | "Distant kin" |
| Dwalin | "Deep sleeper" |
| Gimli | "Fire" |
| Durin | First dwarf (legendary) |
| Thrain | "Yearning" |
| Dain | "Dead" (ironic honor name) |
| Bofur | "Proud" |
| Bombur | "Bulging" |
| Oin | "Lonely" |

### Female Names
| Name | Meaning/Notes |
|------|---------------|
| Dis | "Lady" (only named dwarf-woman in Tolkien) |
| Mim | "Small precious one" |
| Gunnloda | "Battle invitation" |
| Helga | "Holy, blessed" |
| Bruni | "Brown, armored" |
| Kethra | "Kettle, cauldron" |
| Dagni | "New day" |
| Tordis | "Thor's goddess" |

### Clan Names
Dwarf clan names are **compound words** reflecting their craft, home, or legendary deeds:
- **Battlehammer** - Warrior clan
- **Ironforge** - Smithing clan
- **Fireforge** - Smelters
- **Stoneshield** - Defenders
- **Goldhelm** - Wealthy/royal
- **Deepdelve** - Miners
- **Coppermane** - Red-bearded lineage
- **Anvilborn** - Master crafters

### Naming Construction
```
[Harsh Syllable] + [Strong Ending]
THOR + IN = Thorin
DUR + IN = Durin
BAL + IN = Balin
```

## Elven Names

Elven names feature **flowing vowels**, **soft consonants**, and melodic syllables that feel musical when spoken.

### Phonetic Elements
- **Common consonants:** L, N, R, TH, D, S, W, V
- **Common vowels:** Long A, E, I (ae, ea, ia combinations)
- **Endings:** -iel, -wen, -ion, -dil, -or, -il, -ien

## Halfling Names

Halfling names are **simple**, **friendly**, and often have an **Anglo-Saxon** or old English feel.

### Phonetic Elements
- **Simple syllables:** Short, easy to pronounce
- **Familiar sounds:** Like nicknames or diminutives
- **Warm feeling:** Names that sound approachable

### Male Names
| Name | Style |
|------|-------|
| Bilbo | Whimsical |
| Frodo | Wise/thoughtful |
| Samwise | "Half-wise" (humble) |
| Merry | Cheerful |
| Pippin | Young/energetic |
| Tolman | Traditional |
| Bungo | Comfortable |
| Drogo | Adventurous |
| Hamfast | Sturdy |
| Largo | Big (ironic) |

### Female Names
| Name | Style |
|------|-------|
| Rosie | Floral |
| Elanor | Flower name |
| Primula | Nature |
| Belladonna | Mysterious |
| Lobelia | Proper/fussy |
| Daisy | Sweet |
| Marigold | Cheerful |

### Halfling Surnames
Descriptive surnames often relating to:

- **Traits:** Took (bold), Proudfoot, Bracegirdle

## Gnome Names

Gnome names are **long**, **complex**, and often **humorous** with multiple components.

### Naming Convention
Gnomes typically have:
1. A **given name** (often elaborate)
2. A **clan name**
3. A **nickname** (what they actually go by)

### Full Names Examples
| Full Name | Nickname |
|-----------|----------|
| Dimble Nackle Timbers Daergel Nackle | Dimble |
| Pock Stumpybumkin Nackle | Pock |
| Boddynock Glimkettle | Boddy |
| Zanna Cobblelob Scheppen | Zanna |
| Waywocket Pilwicken | Wicket |

### Common Name Elements
- **Syllables:** -wick, -nock, -ble, -dock, -wobble
- **Sounds:** Clicking, bouncing consonants
- **Style:** Playful, inventive, tongue-twisting

### Clan Names
- Beren (precious)
- Daergel (ancient)
- Folkor (folklore)
- Garrick (oak)
- Nackle (clever)
- Scheppen (maker)
- Timbers (carpenter)

## Orcish Names

Orc names are **guttural**, **aggressive**, and often **monosyllabic or two-syllable**.

### Phonetic Elements
- **Harsh sounds:** G, K, GR, UG, AK, URK, OG
- **Guttural vowels:** U, O, A (short and hard)
- **No soft sounds:** Avoid L, S, TH, W

## Usage Tips

1. **Match to culture:** Names should fit the region's cultural inspiration
2. **Consistency:** All members of a clan/family should share naming patterns
3. **Meaning:** Consider if the name has meaning (especially for elves and goliaths)
4. **Pronunciation:** Names should be pronounceable at the table
5. **Nicknames:** Allow for shortened versions for everyday use
6. **Evolution:** Characters might earn new names through deeds
